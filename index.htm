Сайт состоит из 8 разделов:

  1. Общая информация
  2. Теория
  3. Циклический алгоритм
  4. Примеры и задания
  5. Интеллект-карта
  6. Интерактивное изображение
  7. Интерактивный видеоролик
  8. Познавательное

Каждый раздел содержит полезную информацию по теме "Циклические алгоритмы".

Здесь вы найдете теоретическую информацию по теме для знакомства с циклическими алгоритмами (раздел "Теория"). Также вы найдете конкретные примеры написания циклического алгоритма и логику написания (раздел "Циклический алгоритм"). Не остается без внимания возможность увидеть другие примеры циклов и задания (раздел "Примеры и задания"). Вам будет доступен просмотр интеллект-карты с информацией по теме (раздел "Интеллект-карта"). Также вы сможете ознакомиться с интерактивным изображением, содержащим ссылки на дополнительные источники изучения темы (раздел "Интерактивное изображение"). Интерактивный видеоролик, состоящий из презентации по теме "Циклические алгоритмы", позволит вам перейти на дополнительные источники информации, ответить на вопросы, тем самым проверив свои знания (раздел "Интерактивный видеоролик"). Для самых заинтересованных тут тоже найдется дополнительная информация (раздел "Познавательное").
