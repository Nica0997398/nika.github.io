Основы циклических алгоритмов

Что такое цикл?

Цикл – это повторение одной и той же последовательности команд, пока выполняется условие.

Примеры из жизни:

  • Чистить картофелины, пока они не закончатся в мешке
  • Делать 10 приседаний, отсчитывая каждое

Типы циклических структур

  1. С условием выполнения цикла (предусловием)
  2. С критерием завершения (постусловием)
  3. С указанным числом повторений цикла

Цикл с предусловием (цикл-ПОКА)

Пока выполняется условие, программа будет повторять указанные команды.

Порядок выполнения:

  1. Проверка условия
  2. Если условие истинно - выполнение команд тела цикла
  3. Повтор проверки условия

Особенности:

  • Тело цикла может не выполниться ни разу (если условие сразу ложно)
  • Возможно зацикливание (если условие всегда истинно)

Цикл с постусловием (цикл-ДО)

Команды выполняются хотя бы один раз, затем проверяется условие выхода.

Порядок выполнения:

  1. Выполнение команд тела цикла
  2. Проверка условия
  3. Если условие ложно - повтор выполнения

Цикл с параметром

Используется, когда известно точное количество повторений.

Примеры из жизни:

  • Прием курса лекарств (10 дней)
  • Посадка 20 саженцев
  • 5 уроков в день

Реализация циклов в КуМире

Тип цикла Описание Синтаксис Применение
нц для ... кц Цикл с параметром нц для i от 1 до 5
  вывод i
кц 		Когда известно количество повторений
нц пока ... кц Цикл с условием нц пока x > 0
  x := x - 1
кц 		Когда число повторений неизвестно

Базовые алгоритмы с циклами

1. Сумма чисел от 1 до N

алг Сумма чисел
нач
  цел N, S, i
  вывод "Введите N: "
  ввод N
  S := 0
  нц для i от 1 до N
    S := S + i
  кц
  вывод "Сумма = ", S
кон

2. Проверка числа на простоту

алг Простое число
нач
  цел N, i
  лог isPrime
  вывод "Введите N: "
  ввод N
  isPrime := да
  нц для i от 2 до N-1
    если mod(N, i) = 0 то
      isPrime := нет
      выход
    все
  кц
  вывод "Число простое? ", isPrime
кон

Бесконечные циклы и их предотвращение

Пример бесконечного цикла:

нц пока да
  вывод "Бесконечность!"
кц

Как избежать:

  1. Изменять переменную условия в теле цикла
  2. Использовать команду "выход" для досрочного завершения
  3. Проверять корректность условия завершения
